2. Dados Pessoais Coletados

O sistema coleta apenas os dados necessários para o funcionamento adequado do atendimento médico e gestão da fila. Entre eles:

  • Nome completo
  • CPF
  • Data de nascimento
  • Telefone
  • Informações de triagem
  • Classificação de risco
  • Motivo da consulta e sintomas relatados
  • Histórico de atendimentos na unidade
  • Tempo de espera e sala de atendimento
  • Dados de acesso para usuários internos (login e senha)

3. Finalidade da Coleta dos Dados

Os dados são coletados para possibilitar:

  • Gestão da fila de atendimento em tempo real;
  • Triagem adequada conforme classificação de risco;
  • Acompanhamento do paciente durante o atendimento;
  • Geração de relatórios internos para a equipe da unidade de saúde;
  • Autenticação de usuários (colaboradores autorizados);
  • Transparência ao paciente, permitindo visualizar sua posição na fila.

Nenhum dado é coletado sem necessidade. Tudo segue princípios da LGPD (Lei Geral de Proteção de Dados - Lei 13.709/2018).

4. Como os Dados São Armazenados e Protegidos

O sistema adota medidas de segurança como:

  • Armazenamento protegido em servidor seguro;
  • Criptografia das senhas de acesso;
  • Controle de acesso por permissões (somente usuários autorizados visualizam os dados);
  • Registro de atividades para segurança;
  • Atualizações periódicas para prevenção de vulnerabilidades.

5. Compartilhamento dos Dados

Os dados não são vendidos nem compartilhados com terceiros. O acesso é permitido apenas para:

  • Equipe da unidade de saúde;
  • Profissionais autorizados envolvidos no atendimento;
  • Gestores responsáveis pelo sistema.

Nenhum dado é divulgado ao público.

6. Direitos dos Titulares de Dados

O paciente tem direito a:

  • Acessar seus dados pessoais;
  • Solicitar correção de informações;
  • Solicitar exclusão de dados quando cabível;
  • Obter informações sobre o uso dos dados;
  • Revogar consentimentos.

Para exercer seus direitos, o titular pode solicitar pelo e-mail informado.

7. Tempo de Armazenamento dos Dados

Os dados permanecem armazenados pelo período necessário para:

  • Cumprir exigências legais;
  • Garantir histórico clínico da unidade;
  • Atender auditorias e relatórios internos.

Após esse período, podem ser anonimizados ou excluídos.
